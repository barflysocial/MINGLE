# One Match AI v4

Render-ready Node app.

## v4 features
- Separate User Mode and Host Mode
- `/` and `/play` = User Mode
- `/host` = Host Mode
- Realistic mobile-friendly Mingle title screen
- Users cannot type, choose, or regenerate avatar names
- System assigns a clean avatar name automatically
- Avatar names are never repeated in the app registry
- Live camera selfie verification
- Cartoon avatar from live selfie
- Anti-copy/paste mode
- Tap-based multiple-choice onboarding
- One Match AI funnel
- First Spark round
- Mutual chat unlock
- Optional mutual real-photo reveal
